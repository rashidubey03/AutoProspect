# Project Instructions

These instructions must be followed while implementing the AutoProspect project.

## Standing Instructions

- Implement the project phase by phase, starting from Phase 1.
- Keep `Docs/ImplementationPlan.md` aligned with `Docs/PRD.md`.
- Store implementation context for each phase in `Docs/phase_contexts.md`.
- Whenever a new feature is added, create a feature branch in addition to `main`.
- After completing each phase, merge the phase work back to `main` and push `main` to the repository.
- Whenever the user gives another standing instruction, update this file.
